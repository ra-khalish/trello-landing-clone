# Trello Landing Clone
A Trello landing page clone made using HTML, and Tailwindcss.

## Getting Started
1. Clone the repo
```
$ git clone https://github.com/ra-khalish/trello-landing-clone.git
$ cd trello-landing-clone
```

2. Open public files
3. Run index.html file on browser
